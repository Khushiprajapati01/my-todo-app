# My Todo List App

Ek simple aur sundar **Personal Todo List** web application.

![Todo App Preview](https://via.placeholder.com/800x400?text=Todo+App+Screenshot)

## ✨ Features

- Task add karna
- Task complete mark karna (checkbox)
- Task delete karna
- Tasks automatically browser mein save ho jaate hain (refresh karne pe bhi rahte hain)
- Mobile friendly design
- Clean aur modern dark UI

## 🛠️ Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript
- LocalStorage (data save karne ke liye)
